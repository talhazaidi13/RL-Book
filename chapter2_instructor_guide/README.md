**Book:** *Deep Reinforcement Learning for Autonomous Intelligent Systems: Theory, Implementation, and Applications*  
**Chapter 2:** Deep Reinforcement Learning Fundamentals: Algorithm Families, Taxonomy, and Implementation Foundations

This repository contains instructor-facing solutions, small executable examples, and a Colab-style notebook for the Chapter 2 exercises. The materials are designed for teaching assistants, instructors, and authors preparing companion GitHub resources.

## Pedagogical scope

These solutions are not meant to reveal complete implementations of DQN, PPO, SAC, or model-based RL. Chapter 2 is a taxonomy and implementation-foundation chapter. The code therefore focuses on the reusable building blocks students need before the algorithm-specific chapters:

- Q-learning target calculation;
- empirical squared Bellman-error loss;
- soft target-network updates;
- replay-buffer and rollout-buffer structure;
- why old log probabilities matter in on-policy implementation;
- supervised dynamics-model loss;
- task-to-algorithm selection logic.

## Suggested use in teaching

- Assign the textbook exercises without solutions.
- Use `instructor_guide_ch2_solutions.md` as grading support.
- Use the notebook in a recitation or lab session.
- Use `src/ch2_rl_fundamentals/ch2_utils.py` as minimal reference code.
- Extend the code in Chapters 3--6 when full algorithms are introduced.
